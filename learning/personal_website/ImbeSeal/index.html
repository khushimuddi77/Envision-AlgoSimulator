Pathfinding-Visualizer

Try using the different algorithms of Pathfinding


The goal of this project is to create a web based e-Learning tool, 'PathFinding Visualizer', which can be used to visualize shortest path algorithms. The basic idea of this project is for the computer to find a certain path between start to finish which having certain constaints applied through the maze such as a Post, whcih requires the computer to path find from start to the post, and then to the end.
There are multiple logics already present on the website which are given below.

  • Dijkstra's shortest path algorithm
  • A* Search Algorithm
  • Best First Search (Informed Search)
  • Swarm Algorithm
  • Breadth First Search
  • Depth First Search
